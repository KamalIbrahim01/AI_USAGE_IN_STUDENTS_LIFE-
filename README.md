# AI Usage in Student Life Analysis

## Project Overview

This project explores how students use AI assistants in their academic activities and examines the relationship between AI usage patterns, student satisfaction, engagement, and learning outcomes.

Using Excel, Pivot Tables, and Dashboard Visualizations, the analysis identifies trends in student behavior, task completion, and AI adoption across different education levels and disciplines.

---

## Objectives

- Analyze student interaction with AI assistants.
- Evaluate satisfaction and engagement levels.
- Measure task completion outcomes.
- Compare AI usage across student levels and disciplines.
- Identify factors influencing continued AI usage.

---

## Dataset Information

The dataset contains 10,000 student AI interaction records and includes:

- Student Level
- Discipline
- Session Date
- Session Length
- Total Prompts Used
- Task Type
- AI Assistance Level
- Satisfaction Rating
- Final Outcome
- Engagement Score
- Prompt Efficiency
- Repeat Usage Indicator

## Key Insights

- Students frequently use AI tools for academic support and assignment-related tasks.
- Higher engagement scores generally correspond to improved learning outcomes.
- Satisfaction levels influence the likelihood of continued AI usage.
- Usage patterns vary across disciplines and educational levels.
- Prompt efficiency can impact task completion success.
